# TrainLog Frontend (Angular)

## Requisitos
- Node.js 20+
- Backend de TrainLog corriendo en `http://localhost:3000`

## Ejecutar
```bash
cd frontend
npm install
npm start
```

La app se levanta en `http://localhost:4200` y usa proxy para `/users`, `/workouts`, `/exercises`, `/logs` hacia el backend.
